IRC-Escalations.

Greasemonkey script for IRC escalations

Encore Script
CloudControl Script

Greasemonkey is a Firefox Add-on that allows you to modify websites with your own custom scripts. This script will quickly gather hypervisior and slice information from Cloud Control or Encore and will be used to escalate to the Cloud Server Support Escalations team.

In order to properly install this script, click one of the 2 links above, the Encore script is for Encore, the Cloud Control script is for Cloud Control. Once you have the script installed you can then navigate to any server in Encore or Cloud Control and you should see the following.
